Welcome to GitHub Connect4Cpp's Page.

Give it a try

You will be the first to move. Enter 0 to go in the left hand most column, 6 to go in the right hand most column. Your move wont be shown until the autoplayer has made its move.

The game uses a Minimax algorithm with alpha-beta pruning... after watching this lecture

The code has been compiled to JS using EmScripten. It might struggle on phone or tablet
